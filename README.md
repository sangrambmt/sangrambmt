## 🚀 Featured projects

### 🚂 Predictive Maintenance for Railway Equipment
Built LSTM and Prophet models to predict locomotive sensor behaviour (temperature, vibration, brake pressure) **5–30 minutes ahead**, deployed via Azure Data Factory, Databricks and MLflow.
Reduced unexpected equipment failures by **18%** and improved model accuracy by **22%**.

### 💬 Marketing Insights RAG Chatbot
Designed a RAG system with Azure AI Search, text-to-SQL retrieval and a fine-tuned LLM over Azure Data Lake and Azure SQL, with Power BI and Tableau dashboards for AI metrics.
Improved data workflow speed by **40%**, accuracy by **20%**, productivity by **30%**.

### 📅 SmartSales Scheduler — AI-Powered Email & Meeting Assistant
Built an AI agent integrating Gmail APIs, Google Calendar and LLM intent recognition to fully automate outreach and meeting scheduling under Agile execution.
Reduced manual workload by **70%** and increased client meeting conversion by **30%**.

### 📈 Lead Conversion Probability Prediction Model
Developed a predictive ML model using behavioural and demographic data, integrated with Salesforce and Azure ML for real-time lead scoring directly inside the sales workflow.
Boosted marketing ROI by **25%**.
